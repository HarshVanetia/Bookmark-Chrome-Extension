# Chrome Extension with Vite and React

This project is a Chrome extension built using Vite and React. It serves as a template for creating a simple Chrome extension with a popup interface.

## Project Structure

```
chrome-extension-vite-react
├── public
│   ├── manifest.json          # Metadata for the Chrome extension
├── src
│   ├── background.js          # Background script for the extension
│   ├── content.js             # Content script for interacting with web pages
│   ├── popup
│   │   ├── App.jsx            # Main React component for the popup UI
│   │   ├── index.css          # Styles for the popup UI
│   │   └── main.jsx           # Entry point for the React application in the popup
│   └── assets
│       └── torch.png           # Logo or icon for the extension
├── vite.config.js             # Configuration file for Vite
├── package.json                # npm configuration file
├── .eslintrc.js               # ESLint configuration file
└── README.md                  # Documentation for the project
```

## Getting Started

1. **Clone the repository:**

    ```
    git clone <repository-url>
    cd chrome-extension-vite-react
    ```

2. **Install dependencies:**

    ```
    npm install
    ```

3. **Build the project:**

    ```
    npm run build
    ```

4. **Load the extension in Chrome:**
    - Open Chrome and navigate to `chrome://extensions/`.
    - Enable "Developer mode".
    - Click "Load unpacked" and select the `dist` folder generated by Vite.

## Usage

-   Click on the extension icon in the Chrome toolbar to open the popup.
-   The popup will display the UI defined in `App.jsx`.
